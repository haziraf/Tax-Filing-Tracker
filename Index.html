Security & Settings
Open Access
No security configured — anyone with the URL can access.
Authorised Email Addresses
Only listed emails can log in
Leave empty to allow any email (passcode becomes the only gate).
Shared Passcode
All authorised users share the same passcode
Leave blank to keep the existing passcode unchanged. Changing it signs out all current users.
Locked out? Run resetSecurity() in the Apps Script editor to restore open access.
How Access Works / Distributing This App
For teams and selling
Team / Family Access
  1. Add each person's Gmail address to the whitelist
  2. Set a shared passcode and share it privately
  3. Share the web app URL with them
  4. To revoke: remove their email from the list
  5. To revoke everyone: change the passcode
Selling / Distributing
  1. Each buyer deploys their own copy of the script
  2. They set their own email list + passcode
  3. Their data is in their own Drive — fully isolated
  4. Share via GitHub or Apps Script copy link
